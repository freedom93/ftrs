1.1 CSS属性：shape-outside

shape-outside

2017年8月19日,由ROBIN RENDLE最后更新 原文地址

标签：BASE-SHAPE、SHAPE-OUTSIDE

shape-outside属性控制内容将环绕一个浮动元素的限定框。通常这样文本可以回流形状成一个圆,椭圆或一个多边形:

.element {
  float: left;
  shape-outside: circle(50%);
  width: 200px;
  height: 200px;
}

重要的是，要注意这个属性只能在浮动元素中生效,尽管未来可能会改变。shape-outside属性也可以处理transitions or animations。

Values

  • circle(): 绘制圆形
  • ellipse(): 绘制椭圆形
  • inset(): 绘制方形
  • polygon(): 绘制多边形
  • url(): 制定文字环绕的图像路径
  • initial: 默认值（？？the float area is unaffected.）
  • inherit: 可以继承父标签的shape-outside 属性值

以下盒模型的参考值应该用于定位形状:

  • margin-box
  • padding-box
  • border-box

这些值应该附加到最后,例如:shape-outside: circle(50% at 0 0) padding-box。默认使用 margin-box。

ellipse()

.element {
  shape-outside: ellipse(150px 300px at 50% 50%);
}

这样想可能更好:shape-outside属性改变元素与周围的其他元素之间的关系,而不是几何元素的本身。我们需要使用 shape-outside与clip-path()属性修正它,比如在这个例子中:

circle()

.element {
  shape-outside: circle(50%);
}

这个函数创建一个圆,在上面的代码示例中,将创建一个半径是.element一半高度和宽度的圆。circle()函数也可以使用相同的语法在内部定位形状。

url()

.element {
  shape-outside: url('circle.png');
}

在这个例子中,我们有两个浮动图像,每个边上有一个文本块。因为两张图片设置了shape-outside属性，然后下面的文本就会避免浮动。

也可以设置shape-image-threshold属性告诉浏览器一个图片的像素阀值,浏览器根据其透明度创建一个形状。例如:

.element {
  shape-outside: url('image.png');
  shape-image-threshold: 0.5;
}

在这个例子中根据唯一的像素阀值,创建上面的形状将会有50%的透明度。shape-image-threshold属性的值0.0(透明)到`1.0(不透明)都是有效的。

polygon()

.element {
  shape-outside: polygon(0 0, 0 200px, 300px 600px);
}

这个函数创建任何有三个或三个以上顶点的形状,例如:

需要注意的是,如果这个属性将是动画，当你声明动画状态时需要相同数量的顶点:

.element {
  shape-outside: polygon(0 0, 100% 0, 100% 100%, 0 100%);
  transition: shape-outside 1s;
}
.element:hover {
    shape-outside: polygon(0 0, 100% 50%, 100% 50%, 0 100%);
}

inset()

.element {
  shape-outside: inset(100px 100px 100px 100px 10px);
  /* shape-outside: inset(top right bottom left border-radius); */
}
